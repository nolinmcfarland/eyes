# Eyes
Watch them watch you! Animated eyes that follow your cursor.

![Eyes](./images/eyes.png)

## Installation
 
Here's how to get started with Eyes:
 
1. Clone the project!
1. Run `index.html`.
 
## Usage

- As you move your cursor, the eyes will follow!
